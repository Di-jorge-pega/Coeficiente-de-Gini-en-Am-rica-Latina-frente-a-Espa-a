# Coeficiente-de-Gini-en-Am-rica-Latina-frente-a-Espa-a
El coeficiente de Gini resume en un número qué tan repartidos están los ingresos en un país: 0 es igualdad total; 1, un único receptor de todo el ingreso. La línea punteada gris marca la media simple de los 18 países latinoamericanos por año. Selecciona varios países a la vez para comparar sus trayectorias entre ellos y España
